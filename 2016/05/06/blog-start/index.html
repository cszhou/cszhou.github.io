5.Godaddy域名注册

Godaddy支持支付宝，所以过程很简单。Godaddy的优惠码很多，如果没找到的话，域名选好放在购物车后，过一两天Godaddy就会往注册邮箱发优惠码了。
注册域名后，改用dnspod解析域名。
在Godaddy账户管理中，选择注册的域名，然后Manage DNS -> Name Servers选Custom，并增加两条记录，F1G1NS1.DNSPOD.NET和F1G1NS2.DNSPOD.NET
去Dnspod，域名解析栏目下，添加域名，可以看到已有两条NS记录类型。
增加一条A记录，让域名指向你的xxx.github.io网站对应的ip（可以ping得到）。
增加一条CNAME记录，让域名指向xxx.github.io
